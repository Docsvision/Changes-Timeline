# Timeline

Общая страничка изменений в продукте и документации docsvision.

## Технологии

- vue 3
- quasar
- vite
- typescript
- pug

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```
